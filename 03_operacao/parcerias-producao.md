# 🤝 Parcerias de Produção

A Zhy opera com um modelo híbrido: **produção própria** para cápsulas e solúveis + **white-label/co-packer** para café e sucos.

---

## MushSpresso — Café

| Aspecto | Detalhe |
|---|---|
| **Tipo de parceria** | White-label |
| **O que produz** | Café em cápsulas Nespresso, café moído |
| **Base** | Café Arábica Premium + cogumelos (90/10) |
| **Status** | 🟡 Próximo ao lançamento |

A MushSpresso produz e acondiciona o café com a formulação Zhy, que é vendido sob a marca Zhy.

📁 [Documentação Parceria](../08_recursos/Parcería_MushSpresso/)

---

## Vida Líquida — Sucos Cold Press & Shots

| Aspecto | Detalhe |
|---|---|
| **Tipo de parceria** | Produção terceirizada |
| **O que produz** | Sucos prensados a frio com cogumelos |
| **Futuro** | Shots (sucos concentrados menores) |
| **Status** | 🟡 Próximo ao lançamento (sucos) / ⚪ Fase 2/3 (shots) |

📁 [Proposta Vida Líquida](../08_recursos/Parcería_Vida_Líquida/)

---

## Produção Própria — Laboratório Zhy

| Aspecto | Detalhe |
|---|---|
| **Tipo** | Produção própria |
| **O que produz** | Cápsulas suplemento (quando ANVISA aprovar) |
| **Futuro** | Solúveis (necessita novos equipamentos) |
| **Local** | Rio de Janeiro |
| **Status** | Preparado para cápsulas / Em planejamento para solúveis |

---

## Modelo Estratégico

```
Fase 1 (atual):
├── Café → MushSpresso (white-label)
├── Sucos → Vida Líquida (terceirizado)
└── Cápsulas → Produção própria (aguardando ANVISA)

Fase 2:
├── Solúveis → Produção própria (precisa equipamentos)
└── Shots → Vida Líquida

Fase 3/4:
├── Enlatados → Co-packer a definir
└── Proteína → Produção própria ou co-packer
```
